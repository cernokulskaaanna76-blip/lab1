# Лабораторна робота №3  
## Розробка бекенду з використанням SQLite  
### Тема: Сервіс чергувань у лабораторії

---

## 📌 Загальна інформація

У даній лабораторній роботі реалізовано бекенд застосунку для управління чергуваннями у лабораторії.  
Система дозволяє створювати, переглядати, редагувати та видаляти записи про чергування, а також керувати користувачами, розкладами та обмінами змін.

Проєкт розроблено з використанням:
- Node.js
- Express
- TypeScript
- SQLite (без використання ORM)

---

## 🎯 Мета роботи

- Реалізувати збереження даних у SQLite
- Побудувати REST API з повним CRUD
- Реалізувати зв’язки між таблицями
- Додати валідацію даних
- Забезпечити обробку помилок
- Реалізувати фільтрацію, сортування та пагінацію
- Додати JOIN та агрегаційні запити
- Реалізувати спрощену систему міграцій
